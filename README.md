# OS

A small RISC-V operating system.

It supports (very basic):
+ Console output
+ Virtual (Paging) and physical memory managment
+ Interrupts
+ Syscalls
+ Multitasking (IPI, Mutex)
+ Timing

# Building

To build and run the OS type:
> make

# Requirements:

Following (riscv) tools are required:
- RISC-V toolchain
- Qemu
- GDB
